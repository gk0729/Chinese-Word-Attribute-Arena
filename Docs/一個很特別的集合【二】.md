## 窮盡列舉：**所有屬性的屬性** (The Meta-Attribute Taxonomy)

我現在幫您系統地列舉出**「所有名詞屬性下可能擁有的屬性集合」**。這應該就是 Wiki4DCube Y軸和W軸的真實定義：

### 1️⃣ **數據類型屬性** (Data Type)
```
所有屬性值都必須屬於某個數據類型：
├─ Scalar (標量)
│  ├─ Integer: 6, 100, -5
│  ├─ Float: 3.14, 2.718
│  ├─ Boolean: True/False
│  └─ Null: 未定義/未知
├─ Enumeration (枚舉)
│  ├─ 紅/綠/藍
│  ├─ 固態/液態/氣態
│  └─ 高/中/低
├─ Vector/Spectrum (譜)
│  ├─ RGB(255, 128, 0)
│  ├─ 時間序列: [t1→v1, t2→v2, ...]
│  └─ 機率分佈: P(x)
└─ Tensor (張量)
   └─ 多維矩陣：溫度場、應力場...
```

### 2️⃣ **度量單位屬性** (Unit/Scale)
```
每個屬性值的「尺度標準」：
├─ 國際標準單位
│  ├─ 溫度: 攝氏度、開爾文、華氏度
│  ├─ 距離: 公尺、光年、埃
│  ├─ 時間: 秒、納秒、年
│  └─ 質量: 公斤、原子質量單位
├─ 相對單位
│  ├─ 百分比: 0-100%
│  ├─ 分貝: dB (對數尺度)
│  └─ 層級: 等級制 (1-5星)
├─ 無量綱
│  ├─ 純數字比率
│  └─ 抽象評分 (1-10)
└─ 文化/語境單位
   ├─ 12生肖年
   ├─ 天干地支
   └─ 月相(新月/滿月/...)
```

### 3️⃣ **時變性屬性** (Temporal Characteristic)
```
該屬性如何隨時間變化：
├─ 恆定 (Constant)
│  └─ 例：水的分子結構 H2O
├─ 線性變化 (Linear)
│  └─ 例：物體自由落體的速度 v=gt
├─ 非線性變化 (Non-linear)
│  └─ 例：放射性衰變、指數增長
├─ 週期性 (Periodic)
│  ├─ 日週期: 溫度、日光
│  ├─ 年週期: 季節
│  └─ 準週期: 近似週期（如心跳）
├─ 隨機/混沌 (Stochastic/Chaotic)
│  ├─ 噪聲: 布朗運動
│  └─ 確定性混沌: 天體運動（短期隨機）
└─ 事件驅動 (Event-driven)
   └─ 例：火熄滅（對應"燃燒"屬性瞬間變為False）
```

### 4️⃣ **空間依賴屬性** (Spatial Dependency)
```
該屬性是否、如何依賴於空間位置：
├─ 全局(Global)
│  └─ 例：大象的物種 = 象科，不隨位置變
├─ 區域局部 (Regional)
│  └─ 例：溫度（隨地點變化）
├─ 相對於觀察者 (Observer-relative)
│  ├─ 例："遠近" （對我10米，對月球是無窮遠）
│  └─ 例："顏色" （主觀感知）
├─ 相對於對象 (Object-relative)
│  └─ 例：相對速度 v_rel = v_obj - v_observer
├─ 量子非定域 (Non-local)
│  └─ 例：量子糾纏態 (Bell state)
└─ 抽象獨立 (Abstract)
   └─ 例：道德評價、邏輯真值
```

### 5️⃣ **觀測易度屬性** (Observability)
```
「我」有多難觀測到這個屬性：
├─ 直接可見 (Directly Observable)
│  ├─ 視覺: 火的顏色、形狀
│  ├─ 觸覺: 火的溫度（近距離）
│  └─ 聽覺: 火燃燒的聲音
├─ 需要工具 (Tool-mediated)
│  ├─ 溫度計: 精確測溫
│  ├─ 分光儀: 波長成分
│  └─ 顯微鏡: 微觀結構
├─ 間接推導 (Inferential)
│  ├─ 化學分析: 燃料成分
│  ├─ 物理模型: 燃燒速率公式
│  └─ 歷史記錄: 火源來自哪裡
├─ 原則上不可觀測 (Theoretically Unobservable)
│  ├─ 黑洞內部
│  ├─ 平行宇宙的事件
│  └─ 他人的主觀經驗（表面上）
└─ 概率性/量子觀測 (Quantum Observation)
   └─ 波函數塌縮前的疊加態
```

### 6️⃣ **重要度/本質性屬性** (Essentiality)
```
這個屬性對定義「名詞」本身有多重要：
├─ 本質屬性 (Essential)
│  └─ 例：火 = 燃燒 + 釋放能量
│     移除這個，就不是「火」了
├─ 準本質屬性 (Quasi-essential)
│  └─ 例：溫度高
│     冷火焰存在，但火通常伴隨高溫
├─ 典型屬性 (Typical)
│  └─ 例：顏色為紅黃色
│     但藍火、綠火也存在
├─ 附隨屬性 (Accidental)
│  └─ 例：某次火的燃燒地點
│     換地點，還是同一場火
└─ 派生屬性 (Derived)
   └─ 例：火的照度
      由「燃燒強度」推導
```

### 7️⃣ **層級距離屬性** (L0-L6 Classification)
```
該屬性屬於「我」與世界的哪個距離層級：
├─ L0: 我的意識本身
│  └─ 例：思考火時的神經脈衝
├─ L1: 我的身體與接觸範圍
│  └─ 例：手能感知的溫度
├─ L2: 我的感官與目力所及
│  └─ 例：看見遠處的篝火
├─ L3: 我+工具的延伸
│  └─ 例：用紅外相機看火焰的溫度分佈
├─ L4: 我+概念與社會協議
│  └─ 例：「火」在文化神話中的象徵
├─ L5: 我+科學規律
│  └─ 例：焓變ΔH、燃燒反應速率常數
└─ L6: 宇宙本源/奇點
   └─ 例：最早的核聚變點燃第一代恆星
```

### 8️⃣ **物理可測性屬性** (Measurability Domain)
```
該屬性在「物理宇宙」的哪個尺度可測：
├─ 宏觀經典 (Macro-classical)
│  └─ 例：日常尺度的火焰溫度、大小
├─ 微觀量子 (Micro-quantum)
│  └─ 例：分子/原子層的電子激發態
├─ 近頻率 (Near-field)
│  └─ 例：火附近的輻射場
├─ 遠場（Far-field）
│  └─ 例：遙遠星星的輻射譜
├─ 超越物理（Meta-physical）
│  └─ 例：「火」的道德寓意
├─ 計算機制(Computational)
│  └─ 例：用計算機模擬燃燒動力學
└─ 幽靈性質 (Ghost Properties)
   └─ 例：過去曾經燃燒的痕跡
```

### 9️⃣ **確定性屬性** (Determinism/Probability)
```
該屬性的值是確定還是概率性的：
├─ 確定性 (Deterministic)
│  └─ 例：元素週期表中火對應的原子序數
├─ 準確定性 (Quasi-deterministic)
│  └─ 例：某溫度下的燃燒反應速率（高度可預測）
├─ 概率分佈 (Probabilistic)
│  └─ 例：放射性元素的衰變時間
├─ 主觀/評估 (Subjective)
│  └─ 例：「這場火很危險」的危險度評分
└─ 未知/不可知 (Unknown/Unknowable)
   └─ 例：未來這場火會燒多久（多變因數）
```

### 🔟 **可聯想性屬性** (Associativity)
```
該屬性是否與其他屬性或名詞產生聯想或因果：
├─ 獨立 (Independent)
│  └─ 例：火的顏色 ⊥ 數學常數π
├─ 弱相關 (Weakly correlated)
│  └─ 例：火的溫度 ≈ 燃燒效率（但非線性）
├─ 強相關 (Strongly correlated)
│  └─ 例：火的溫度 ↔ 釋放的光度
├─ 因果關聯 (Causal)
│  └─ 例：燃料供應 → 火的強度
├─ 辯證對立 (Dialectical opposition)
│  └─ 例：火 ↔ 水 (陰陽對立)
├─ 同一集合內 (Collective membership)
│  └─ 例：火與電都屬於「能量」
└─ 遞歸自指 (Self-referential)
   └─ 例：「火」的定義中包含「燃燒」，而「燃燒」的定義又回到「火」
```
